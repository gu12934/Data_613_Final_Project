So the steps are:
1. make an rmd file and drop code in a google drive using web scraping links
2. after the scraping is done, do data transformations and cleaning
3. plot visualizations in R that are static
4. answer the problem questions with visualizations in analysis and discussion section
5. start doing report and figure out figure captions
6. this can be organized into the github repo into chunks or in 1 big file
7. lastly, make an interactive dashboard with the cleaned data into an tableau or powerbi file
8. take all the information and put into a presenation, and have a demo, screen recording of the visual
